Date and time formatting

New Year's Eve 2016 was {{fromDate}}.

This plugin provides date and time formatting capabilities. It can also provide date operations like determining the difference between two dates.

All date and time functions are localized if the library is initialized with the correct locale like this: 

gromit.setLocale('en-US');

The locale is often set to make sure it matches the locale of the server. If this function isn't called then the plugin will use the current browser locale. 

var now = new Date().getTime();
$scope.fullDateTime = gromit.fullDateTimeFormat(now);
$scope.fullDate = gromit.fullDateFormat(now);
$scope.shortDateTime = gromit.shortDateFormat(now);
$scope.shortDate = gromit.shortDateFormat(now, true);
$scope.fromDate = gromit.fromNowFormat(1451606400000);
