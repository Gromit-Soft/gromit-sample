Anchor with no extra pound sign

This anchor won't add an extra point sign to browser location

{{alert.msg}}

All anchors should have an href attribute so the anchor can take keyboard focus, but they should consume the click event so you don't add a pound sign or some other extra characters to the browser location.

This directive will add an href attribute to all anchors that need one and make sure to consume the click so nothing extra gets added to the browser location bar. 

<a ng-click="myFunction">a link</a>
